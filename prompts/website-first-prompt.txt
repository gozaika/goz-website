## 3) Monorepo Architecture Requirements

Create a structure that supports independent deployment of each app:

- `apps/website` (this phase only)
- shared code in reusable packages (no duplication), e.g.:
  - `packages/db` (Supabase client and shared DB helpers)
  - `packages/logger`
  - `packages/config`
  - `packages/utils`
  - `packages/types`
