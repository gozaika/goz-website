# goZaika Next.js Content Package

## Brand Positioning

goZaika is a premium-access mystery meal platform.

It is not:

- a discount app
- a leftovers marketplace

It is:

- discovery-led
- trust-led
- pickup-first

## Messaging Pillars

- Premium taste. Smarter access.
- Discovery, not discounting.
- Mystery in the dish. Clarity in the details.
- Pickup-first, same-day, trust-led experience.
